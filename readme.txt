2024/11/7
尝试用TIM5更新中断来进行拨弹盘推弹的速度的控制
结果：成功，无bug
补充：通过控制ARR的值来控制推弹的速度，记得一定要开启APRE(自动重载预装载功能)，因为如果在没有开启APRE的情况下，大周期把ARR改小，定时器虽然会立刻写入ARR寄存器
	  但是不会立刻开启下一个周期，若是此时CNT还未到小周期的ARR，那么可以正常工作，若是CNT已经大于了小周期的ARR，那么只有等CNT计数溢出了才会开启下一周期
